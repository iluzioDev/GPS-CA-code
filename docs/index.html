Introduction üìã¬∂

GPS L1C/A (Coarse/Acquisition) are sequences of zeroes and ones used in GPS communications, being P-Code other type of code GPS uses.

They are also known by PRN codes (Pseudorandom Noise), but in reality they are generated using an algorithm, which is freely available, and that‚Äôs the main goal of this repo.

Inside this repo is an implementation of the C/A Algorithm developed in Python, so anyone could generate codes with customized polynomials and sequences of n desired bits.
